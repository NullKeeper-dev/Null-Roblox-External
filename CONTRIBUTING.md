# Contributing to Null Roblox External

First off, thank you for considering contributing to this educational project!

## Code of Conduct

This project and everyone participating in it is governed by the [Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code.

## Educational Purpose

Please remember that this project is strictly for educational purposes. Contributions that focus on malicious intent, commercial exploitation, or bypassing anti-cheat for unfair advantages will be rejected.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issue tracker.
- Describe the bug and include steps to reproduce.
- Mention your Windows version and any specific game conditions.

### Suggesting Enhancements
- Open an issue with a clear description of the feature.
- Explain why this would be a valuable addition for educational learning.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. Ensure the code compiles without warnings.
3. Follow the existing coding style (C++20, consistent indentation).
4. Update the README or documentation if necessary.
5. Provide a clear description of the changes in your PR.

## Development Rules
- **No Malicious Payloads:** Any PR containing harmful code will result in an immediate ban.
- **Clean Code:** Use meaningful variable names and comment complex logic.
- **Safety First:** Ensure any kernel-mode changes are well-documented and safe to execute in a test environment.

## Conventional Commits
We prefer the use of conventional commits:
- `feat: ...` for new features
- `fix: ...` for bug fixes
- `docs: ...` for documentation changes
- `refactor: ...` for code restructuring
